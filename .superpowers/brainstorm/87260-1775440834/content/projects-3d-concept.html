How this works in the real implementation

  • R3F + drei springs — real 3D scene with spring-damped camera, not CSS transforms
  • Research Commander expands — when active, its plane opens up to reveal live layers: the dashboard, agent pipelines visualized as flowing data, architecture diagram as 3D nodes
  • arch-drift gets a terminal replay — animated terminal showing init → detect → check → catch violation in real-time
  • Claude Notifier gets live blinking — simulated Kitty tab bar with real animation states cycling
  • Mobile apps get device frames — video playing inside a floating phone in 3D space with UI layers that peel apart on hover
  • Ambient particles + depth fog — the void between projects has life
  • Mouse parallax on everything — subtle depth response even when idle
  • Each project's plane has its own color temperature — warm orange for Commander, green for arch-drift, purple for Notifier, pink for Anime AI, blue for Atlas
