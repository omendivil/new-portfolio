EXTRA

In a program that accepts only the top 10% of Apple's workforce, Omar Mendivil served as a Perception Triage Engineer, leading weekly triage meetings with the Product Integrity team.

He analyzed radar data, communicated issues to engineers, and helped the QE team better understand system behavior — reducing unnecessary radars and making testing more targeted.

Most notably, he identified a recurring issue pattern across multiple radars that helped engineers isolate and fix a critical perception bug.

How It Works

Theme selector buttons sit near the experience section. Clicking one morphs the entire presentation — frame, typography, colors, layout — while the data stays the same. Each theme uses only CSS class swapping + different font loading. Zero layout shift, instant switch.
