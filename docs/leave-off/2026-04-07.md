## Creative Direction (Approved, Not Yet Executed)

### The Animus Concept
Each project gets its own universe. Reference: Assassin's Creed Animus loading.

1. **Base state** — digital void
2. **Project selected** — particles rush together, world solidifies, project UI assembles from the effect
3. **Per-project worlds** (need Midjourney video loops or Blender renders):
   - Research Commander: command center, holographic data streams, orange/amber
   - arch-drift: structural/geometric construction, green
   - Claude Notifier: terminal void, code rain, purple bioluminescence
   - Anime AI: vibrant stylized world, neon cityscape, pink
   - Atlas Chat: connected constellation, message paths, blue
4. **Switching** — current world dematerializes, new world materializes
