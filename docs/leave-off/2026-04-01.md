## Key Design Decisions Made
- **Horizontal, not vertical** — Omar wants it to feel like walking through rooms, not scrolling a page
- **Click-only navigation** — no free scrolling between sections. Users must click exit buttons or nav items.
- **Terminal commands as brand** — the nav already looks like a terminal path (`~/omar · home · projects`), so terminal transitions reinforce the developer identity
- **Each section builds itself on entry** — content assembles piece by piece, not just fades in
- **Sections stay mounted** — performance optimization, no unmount/remount on navigation
